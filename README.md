Финансовые термины:
счет → invoice (счёт на оплату) / account (банковский счёт)
упд → UPD (Universal Transfer Document - официальный российский термин) / universal transfer document
универсальный передаточный документ → universal transfer document (UTD) - заменяет счёт-фактуру и накладную
акт выполненных работ → certificate of work completion / act of work performed
акт оказанных услуг → certificate of services rendered / act of services provided

Учётные термины:
номенклатура → nomenclature / product catalog / item list
количество → quantity (QTY)
цена → price / unit price
стоимость → cost / amount / total value

Дополнительные варианты в контексте:
счёт-фактура → invoice (VAT invoice)
налоговая накладная → tax invoice
товарная накладная → delivery note / bill of lading
платёжное поручение → payment order
авансовый отчёт → advance report
кассовый чек → cash receipt / sales receipt

Примеры использования:
"Выставить счёт" → "Issue an invoice"
"Подписать акт" → "Sign the certificate"
"Номенклатура товаров" → "Product nomenclature" / "Item catalog"
"Стоимость партии" → "Batch cost" / "Lot value
